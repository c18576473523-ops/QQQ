# 钦钦的个人知识库

这是我的个人知识库，用于整理和存储各种知识。

## 目录结构

- 📁 `docs/` - 文档资料
- 📁 `notes/` - 个人笔记  
- 📁 `projects/` - 项目文档
- 📁 `resources/` - 资源收集
- 📁 `archive/` - 归档文件

## 使用指南

1. 使用 Obsidian 打开此文件夹进行笔记管理
2. 使用 Markdown 格式编写内容
3. 定期整理和归档

## 标签系统

- `#工作` - 工作相关
- `#学习` - 学习笔记
- `#技术` - 技术文档
- `#生活` - 生活记录
- `#灵感` - 创意想法
